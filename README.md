# CrimsonSteal Discord Bot

A professional verification + welcome + ticket + autorole bot.

## Setup

1. Install dependencies:
```
npm install
```

2. Create a `.env` file:
```
TOKEN=YOUR_DISCORD_BOT_TOKEN
```

3. Start bot:
```
npm start
```

## Features
- Verification system
- Auto role
- Welcome message
- Ticket system
- Slash commands
- 24/7 Railway support
